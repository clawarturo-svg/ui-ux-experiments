# Reusable Landing Patterns

## Hero patterns
- Giant editorial headline + one-paragraph explanation
- Small eyebrow + strong headline + dual CTA
- Product statement + proof strip under hero

## Section patterns
- 3-card feature grid
- Principles grid
- Split section (essay left / highlight right)
- Single CTA panel with strong contrast

## Visual patterns
- Glass cards
- Soft border + strong shadow
- Minimal pill navigation
- Atmospheric background animation

## UX checks
- CTA visible without scrolling on desktop
- no horizontal overflow on mobile
- hero copy readable in 2-3 breaths
- card spacing consistent
- background animation must not reduce readability
